# UI/UX is Not Design — It’s Psychology

Good design is not about colors and fonts.

It’s about **how users feel**.

## 🧠 Core Principle

Users don’t read.
They scan.

## 🔥 Key Rules

- Keep it simple
- Reduce friction
- Guide attention

## ⚡ Example

Bad UX:
- Too many buttons
- Confusing layout

Good UX:
- Clear hierarchy
- Minimal actions

## 🎯 Golden Rule

> Don’t make users think.

## 🧩 Final Thought

Design is not what it looks like.
It’s how it works.
