# Docker: Build Once, Run Anywhere

Docker has revolutionized how applications are built and deployed.

## 📦 What is Docker?

Docker allows you to package your application with all dependencies into a **container**.

## 🔥 Benefits

- Consistent environments
- Easy deployment
- Lightweight compared to VMs

## ⚙️ Example

```bash
docker build -t my-app .
docker run -p 3000:3000 my-app


🚀 Why Developers Love Docker
